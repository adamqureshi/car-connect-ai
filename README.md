# CarConnect AI MVP Landing Page

Files included:
- `index.html`
- `styles.css`
- `thanks.html`

## Form handling options
This page is wired for Netlify Forms:
- `name="dealer-contact"`
- `data-netlify="true"`
- `action="/thanks.html"`

If you prefer Formspree or another form backend, replace the `<form>` tag in `index.html` with your endpoint.
